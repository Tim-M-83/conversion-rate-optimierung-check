# Assets

Dieses Verzeichnis enthält die lokal verwendeten Bilddateien für den Conversion-Rate-Optimierung-Check.

Die Dateien sind bewusst mit URL-sicheren Namen abgelegt, damit die statische Website zuverlässig über GitHub Pages funktioniert:

- `conversion-rate-optimierung.png`
- `conversion-rate-optimierung-dienstleister.png`
- `conversion-rate-optimierung-onlineshop.png`
- `mediencenter-essen-adler.png`

Alle Bilder werden lokal eingebunden. Es werden keine externen Bildquellen oder APIs benötigt.
